# Клуб будущих миллионеров

Плоская версия многостраничного сайта для GitHub Pages.

## Структура
- `index.html`
- `about.html`
- `path.html`
- `materials.html`
- `faq.html`
- `style.css`
- `main.js`
- `logo.png`

## Telegram
Ссылка на Telegram-канал задаётся в `main.js` в переменной `TELEGRAM_URL`.
