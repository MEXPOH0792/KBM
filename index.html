О проекте

Этот проект создан для мусульман, которые хотят разобраться в деньгах без крайностей, давления и сомнительных решений.

Здесь собраны базовые материалы по исламским финансам: как смотреть на доходы, расходы, накопления и инвестиции так, чтобы это было понятно, практически полезно и не противоречило шариату.

  • Простым языком — без искусственно сложных объяснений
  • С опорой на принципы — без сомнительных трактовок
  • С пользой для жизни — чтобы знания можно было применять
Для кого это Для тех, кто только заходит в тему, и для тех, кто хочет собрать основу заново без лишнего давления.
Как устроены материалы Короткие тексты, разборы частых вопросов и простые примеры из повседневной жизни.
Задача проекта Не подталкивать к быстрым решениям, а помочь спокойно понять, на что вообще опираться.

Сначала нужно понять основу. Потом — привести в порядок личные финансы. И только после этого имеет смысл переходить к инструментам.
